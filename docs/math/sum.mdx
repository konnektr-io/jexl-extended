# sum

Calculates the sum of an array of numbers.

**Type:** transform

## Parameters

- **input** (array): The input array of numbers or individual number arguments.

## Returns

**Type:** `number`

The sum of all numbers, or NaN if input is not an array.

## Examples

```javascript
sum([1, 2, 3, 4]) // 10
```

```javascript
[1.5, 2.5, 3.0]|sum // 7
```

```javascript
sum(1, 2, 3, 4) // 10
```
