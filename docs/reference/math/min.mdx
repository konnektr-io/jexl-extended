# min

Finds the minimum value in an array of numbers.

**Type:** transform

## Parameters

- **input** (array): The input array of numbers or individual number arguments.

## Returns

**Type:** `number`

The minimum value, or NaN if input is not an array.

## Examples

```javascript
min([1, 5, 3, 2]) // 1
```

```javascript
[10, 20, 15]|min // 10
```

```javascript
min(1, 5, 3, 2) // 1
```
