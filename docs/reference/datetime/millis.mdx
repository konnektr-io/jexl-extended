# millis

Returns the current date and time in milliseconds since the Unix epoch.

**Type:** function

## Returns

**Type:** `number`

The current timestamp in milliseconds.

## Examples

```javascript
millis() // 1703505000000
```

```javascript
millis() // 1703505123456 (different time)
```
