# abs

Returns the absolute value of a number.

**Type:** transform

## Aliases

`absoluteValue`, `abs`

## Parameters

- **input** (unknown): The input number to get the absolute value of.

## Returns

**Type:** `number`

The absolute value, or NaN if input cannot be converted to a number.

## Examples

```javascript
absoluteValue(-5) // 5
```

```javascript
(-10)|absoluteValue // 10
```

```javascript
absoluteValue(3.14) // 3.14
```
