# Utility

General utility functions

## Functions

- [`case`](./case): Evaluates a list of predicates and returns the first result expression whose predicate is satisfied.
- [`eval`](./eval): Evaluates a JEXL expression and returns the result.
- [`length`](./length): Returns the number of characters in a string, or the length of an array.
- [`not`](./not): Returns the logical NOT of the input.
- [`uuid`](./uuid): Generates a new UUID (Universally Unique Identifier).
