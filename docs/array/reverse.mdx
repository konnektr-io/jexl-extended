# reverse

Reverses the elements of an array.

**Type:** transform

## Aliases

`arrayReverse`, `reverse`

## Parameters

- **input** (array): The input values to reverse.

## Returns

**Type:** `array`

A new array with elements in reverse order, or empty array if no valid input.

## Examples

```javascript
reverse([1, 2, 3]) // [3, 2, 1]
```

```javascript
[1, 2, 3]|reverse // [3, 2, 1]
```

```javascript
reverse(["a", "b", "c"]) // ["c", "b", "a"]
```
