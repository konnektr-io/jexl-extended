# uuid

Generates a new UUID (Universally Unique Identifier).

**Type:** function

## Aliases

`uid`

## Returns

**Type:** `string`

A new UUID v4 string.

## Examples

```javascript
uuid() // "123e4567-e89b-12d3-a456-426614174000"
```

```javascript
uuid() // "987fcdeb-51a2-43d7-b123-456789abcdef" (different each time)
```
