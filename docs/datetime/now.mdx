# now

Returns the current date and time in the ISO 8601 format.

**Type:** function

## Returns

**Type:** `string`

The current date and time as an ISO 8601 string.

## Examples

```javascript
now() // "2023-12-25T10:30:00.000Z"
```

```javascript
now() // "2023-12-25T14:45:30.123Z" (different time)
```
