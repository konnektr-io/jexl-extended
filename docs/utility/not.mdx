# not

Returns the logical NOT of the input.

**Type:** transform

## Parameters

- **input** (unknown): The input to apply logical NOT to.

## Returns

**Type:** `boolean`

The logical NOT of the input converted to boolean.

## Examples

```javascript
not(true) // false
```

```javascript
false|not // true
```

```javascript
not(0) // true
```
