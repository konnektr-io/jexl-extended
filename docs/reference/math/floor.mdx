# floor

Rounds a number down to the nearest integer.

**Type:** transform

## Parameters

- **input** (unknown): The input number to round down.

## Returns

**Type:** `number`

The rounded down integer, or NaN if input cannot be converted to a number.

## Examples

```javascript
floor(3.7) // 3
```

```javascript
(3.14)|floor // 3
```

```javascript
floor(-2.8) // -3
```
