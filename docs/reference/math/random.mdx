# random

Generates a random number between 0 (inclusive) and 1 (exclusive).

**Type:** function

## Aliases

`randomNumber`

## Returns

**Type:** `number`

A random floating-point number between 0 and 1.

## Examples

```javascript
randomNumber() // 0.123456789 (example output)
```

```javascript
randomNumber() // 0.987654321 (different each time)
```
